ChatGPT Takeout

Export your ChatGPT conversations & images to a self-contained HTML archive — all in your browser.

1 · Proxy URL using official proxy

ChatGPT blocks browser requests (CORS). Deploy the tiny Cloudflare Worker from this repo (free, 60 seconds) and paste the URL below. If you don't want to self-host, see the privacy notes.

Use my own proxy instead

2 · Paste cURL

  1. Open chatgpt.com and sign in.
  2. Open DevTools → Network tab.
  3. Reload the page. Find a request to /backend-api/conversations.
  4. Right-click → Copy → Copy as cURL.
  5. Paste the whole thing into the box below.

Idle.
